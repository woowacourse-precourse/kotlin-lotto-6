# 🎰 로또
**일확천금**을 노리는 사람은 로또를 구매하고, 당첨번호를 확인하여 수익률까지 알 수 있는 프로그램

## 🗒️ 기능
- 로또 구매
- 로또 방송 MC가 로또 번호 및 보너스 번호 뽑음
- 로또 당첨 번호를 확인하고, 당첨 여부 및 수익률을 알 수 있음
