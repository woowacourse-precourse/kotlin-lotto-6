# 📚 3주차 미션 - 로또

## 구현할 기능 목록
### 공통
- 입력
- [x] 잘못 입력시 다시 입력받기
- 출력
- [x] 예외처리(IllegalArgumentException)
- [x] 에러 메세지 `[ERROR]` 붙여서 출력

### 1. 로또 구입해서 발행하기
- 입력
- [x] 1000 단위의 숫자
- 출력
- [x] 구매한 개수만큼 로또 발행

### 2. 당첨 번호와 보너스 번호 입력받기
- 입력
- [x] 쉼표로 구분된 중복없는 6개 숫자
- [x] 보너스 번호 1개 입력받기

### 3. 당첨 결과 출력하기
- 출력
- [x] 각 등수마다 당첨 개수 출력
- [x] 수익률 계산 후 출력
