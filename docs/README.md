# 🚀기능 요구 사항

## 🚀입력
+ 사용자
  + 구입 금액
  + 당첨 번호
  + 보너스 번호

## 🚀출력
+ 메세지
  + 구입 금액 입력
  + 구매 개수 출력
  + 당첨 번호 입력
  + 보너스 번호 입력
  + 당첨 통계
  + 총 수익률
+ 로또 수량
+ 각 로또 번호
+ 당첨 내역
+ 수익률

## 🚀구현 사항
+ 각 개수별 로또 번호 일치/불일치 판별
+ 수익률 계산
+ 수익률 소수 둘째 자리에서 반올림

## 🚀유효성 검사
+ "[ERROR]"로 시작하는 에러 메시지를 출력 후 그 부분부터 입력을 다시 받는다.
+ 로또 번호
+ 구입 금액
+ 보너스 번호
+ 각 항목 중복, 타입, 공백 등 고려

## 🚀데이터
+ 당첨 번호
+ 구입 금액
+ 보너스 번호
+ 로또 수량
+ 각 발행한 로또 번호
+ 당첨 내역
+ 수익률
