## 🧩 추가 요구 사항

```- indent depth가 3이 넘지 않도록 한다. (2까지 허용)```
<br>

```- 함수의 길이가 15라인을 넘어서면 안된다.```
<br>

```- 함수를 구현할 때 SRP를 잘 지키도록 구현.```
<br>

```- else를 지양한다. (early return 사용)```
<br>

```- if/else 가 when보다 더 깔끔해 보일 수도 있음. 본인의 판단하에 때에 맞게 사용.```
<br>

```- Enum 클래스를 사용해본다.```
