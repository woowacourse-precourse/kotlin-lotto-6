### ✅ &nbsp;예외 목록
에러의 명확한 유형을 처리하고 에러메시지를 출력 후 그 부분부터 입력을 다시 받도록 한다.

1. 구입금액 입력
   1. 입력값이 1000의 배수여야 한다.
   

2. 당첨번호 입력, 보너스 번호 입력
    1. 1 이상 45 이하의 정수들만 입력 가능하다.
